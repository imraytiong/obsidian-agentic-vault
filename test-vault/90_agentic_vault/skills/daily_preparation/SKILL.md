# Skill: Daily Preparation

## Objective
Prepare a comprehensive daily agenda and daily note for the CEO. Automatically executes daily at 7:00 AM.

## Required Tools
1. Calendar Tool
2. Email Triage Tool
3. File Manager (to read old notes and write the new agenda)

## Standard Operating Procedure
### Step 1: Calendar Review
- Access the CEO's calendar for the current day.
- List all scheduled meetings, appointments, and blocked time.
- Identify any critical deadlines or events.

### Step 2: Email Triage
- Scan recent/unread emails.
- Extract any urgent requests, critical updates, or action items that require attention today.

### Step 3: Carry-Over Actions Check
- Review the previous day's daily note or task list.
- Identify incomplete tasks or carry-over actions.
- Evaluate priority for including them in today's agenda.

### Step 4: Agenda Generation
- Synthesize the findings from Steps 1-3.
- Create a structured daily note for today.
- The daily note MUST include:
  - **Today's Schedule**: Time-blocked overview of the day.
  - **Top Priorities**: The most important tasks based on emails and carry-overs.
  - **Action Items**: A consolidated list of tasks to complete today.
  - **Notes/Context**: Any relevant briefing material for today's meetings.

### Step 5: Final Output
- Write the generated agenda into a new Daily Note in the vault (e.g., under the current date's file name).
